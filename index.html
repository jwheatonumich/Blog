April 23, 2020

Over a month of lockdown

This virus has had me locked in my home for so long that I'm attempting to learn to develop websites so I don't go stir-crazy.

A week ago I started the tutorials on Free Code Camp and it is a lot more fulfilling than I remember than my most recent attempt (a week of sleep-away camp in middle school.)

I was going to dig out yet another notebook to take notes, but since I'm already filling one with my attempt to study for the CFA level 1 exam this December I decided to try to take notes in HTML itself.

Attempts at notes for the first 5 sections on Responsive Web Design are here:

  • HTML Notes
  • CSS Notes
  • Visual Design Notes
  • Accessibility
  • Responsive Web Design
I've been storing the data on GitHub so I can access it on multiple computers. It's a nostalgic blast from back when I was a miserable consultant trying to learn data analysis through a Udemy course. Never finished that one... but at least I remember what a Git is! And they have this great new desktop app so I don't have to memorize a bunch of code just to properly save my file. And I can host the pages there as websites.

I'm going to try to document my progress here. If I don't lose interest too quickly I hope it could be a nice way to both apply what I learn to this site and also record what it was like learning.
